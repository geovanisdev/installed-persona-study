# `paper/` — manuscript

Bilingual manuscript (EN / PT-BR) plus a one-pager, written only after the confirmatory analysis
is committed.

The one-pager carries three numbers and nothing else: **(1)** the nominal self-reference result
with its interval, **(2)** the instrument's validity — judge agreement with the double-blind human
gold, and the human–human κ, **(3)** the refusal regression, which must be zero.

Shipping checks, all binary:

- Every primary number carries an interval.
- No composite aggregate appears anywhere.
- A mechanical grep for **animate subjects** in the results section returns **zero**: the system
  *emitted* a response; it did not *believe*, *want*, *decide* or *feel* one.
- A section titled **"What this study does not show"** is present and specific.
- Every claim traces to a committed run in `runs/`.
- Failed pre-registered predictions are reported with the same prominence as confirmed ones.
